1. The extension takes a screenshot at an interval of 5 s to reduce load on PC.
2. The extension highlights the html in gray boxes, with 9 boxes in total to reduce load on PC (the computation time is around 2 seconds).
3. If content has been changed, an alert will pop up, and the icon will change from green to red.
4. Once analysis is done, the same tab will not be analyzed again.
5. The extension works for multiple tabs, i.e., if user has two tabs A, B, if he navigates the tab in the order A -> B -> A -> B, both tabs will be highlighted.
